1) Если сделать XOR для двух строк в кодировке ASCII "Привет, мир!" и "Привет, мир!", какой будет результат?
    Любой байт XOR сам с собой = 0.
    Пример: 01101001 XOR 01101001 = 00000000
    Ответ результат будет 0

2) Приведи пример двух бинарных строк длиной 4 символа, которые при операции XOR друг с другом дадут одну из исходных.
Пример:
    Строка A: 1111
    Строка B: 0000
    A XOR B = 1111

3) Реши данное выражение (напиши, что останется после выполнения всех операций XOR):
(A XOR B) XOR A
    Ответ: (A XOR B) XOR A = B
